# MedAssist AI — Knowledge Base

Medical knowledge base for the MedAssist AI Q&A chatbot. All content is in English, follows WHO factsheet format, and is intended for educational purposes only.

> **Disclaimer:** This knowledge base is for informational and educational purposes only. It does not constitute medical advice, diagnosis, or treatment. Always consult a qualified healthcare professional for medical concerns.

## Summary

- **Total files:** 40 disease/condition files across 8 categories
- **Format:** Markdown with YAML frontmatter
- **Source:** WHO fact sheets and guidelines
- **Language:** English
